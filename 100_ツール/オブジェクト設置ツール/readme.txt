================================
= 制作環境
================================

・使用言語：C++
・開発環境：DirectX9

================================
= 注意事項
================================

・ソリューションではx86での起動をお願いします。

-------------------------------
- 操作
-------------------------------

・移動
[W][A][S][D]キー

・減速移動
[LSHIFT]キー + [W][A][S][D]キー

・上下移動
 [LSHIFT]キー + [R][F]キー
 
・モデルの向き移動（15度）
[T][G]キー

・モデル種類の変更
[TAB]キー

・オブジェクトカテゴリの変更
[CTRL]キー

・オブジェクト設置
[SPACE]キー

・保存
[F5]キー

-------------------------------
- 説明
-------------------------------

Xファイルのオブジェクトを設置して外部ファイルに保存するツールです。
C++での初制作作品のときにマップ生成簡略化のため制作しました。

オブジェクトカテゴリを変更し、接触可能のオブジェクトの設置や敵の設置などが行えます。
各種オブジェクトファイルを読み込むことで設置ができます。
