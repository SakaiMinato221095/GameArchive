-------------------------------
- ゲーム内容
-------------------------------

基本は最大四人でのパーティーゲームです。
マップからパーツを集め、ロケットを修理して脱出するのが目的のゲームです。
途中で妨害してくる敵が出てくるので、ボックスから武器を入手して敵を倒せます。
ロケットを直すことを第一に敵を倒しながら生き残って脱出しましょう。

-------------------------------
- 担当箇所
-------------------------------

タイトル
ランキング
プレイヤーの情報UI
モーションの分割化

-------------------------------
- アピールポイント
-------------------------------

モーションの分割化を実装し、上半身は射撃・下半身は移動などの複数の表現ができるように実装しました。
ランキングでは敵を倒した数や味方を殴った数などパーティー感が出る詳細の値が出るように実装しました。
